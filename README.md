# Trade Market

## Overview
Simple C# solution that provides data access, business logic, and a Web API.

## Repository structure
- `Trade Market.sln` \- solution file  
- `Business/` \- business logic, services, models, interfaces  
- `Data/` \- EF Core DbContext, entities, repositories, migrations  
- `WebApi/` \- ASP.NET Core Web API project and controllers
